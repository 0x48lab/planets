🏭 機械の建築と使用

パッケージャー

用途: ほぼすべてのアイテム作成

構造: 9層 + 看板層

必須: 白樺の看板

使い方:

  1. 看板をクリック
  2. レシピを選択
  3. 材料をバレルに入れる
  4. 看板を再度クリック

空気圧縮機

用途: 空気キャニスターの充填

構造: 4層構造

必須: 白樺の看板

使い方:

  1. 空のキャニスターをチェストに入れる
  2. 1スロット1個ずつ配置
  3. 看板をクリック
  4. 「Compressing...」を確認

精製機

用途: 燃料キャニスター作成

構造: 9層構造

材料: コーラスフルーツ

使い方:

  1. コーラスフルーツをチェストに入れる
  2. 看板をクリック
  3. 精製完了を待つ
  4. 燃料キャニスターを取り出す

アップグレードシステム

アップグレード方法

  1. 宇宙服を装備する
  2. アップグレードキットを持つ
  3. 金床をShift+左クリック
  4. Tier 1→2→3→4の順番で強化

アップグレード種類

  • 🌬️ 空気タンク (Tier 1-4)
  • ⛽ 燃料タンク (Tier 1-4)
  • ☢️ 放射線防護 (Tier 1-4)
  • 🌡️ 温度耐性 (Tier 1-4)

🚀 ロケットと発射

ロケット建造

建造ルール

  • 最大高さ: 15ブロック
  • 発射台の範囲内
  • チェスト必須（燃料用）
  • オークの看板必須

発射手順

  1. 発射台建設

    11×11の鉄・金・ラピスブロック

  2. ロケット建造

    発射台の上に15ブロック以内

  3. 看板設置

    [rocket]
    x座標,y座標,z座標

  4. 燃料投入

    チェストに燃料キャニスター

  5. 発射！

    看板をクリックして目的地選択

⚠️ サバイバルのコツ

酸素管理

予備キャニスターを3個以上携帯。常に酸素ゲージを監視。

隕石対策

地下10ブロック以上に避難。石の天井3層で防御。

放射線対策

適切なTierの防護装備必須。短時間探索を心がける。

緊急脱出

ロケット位置を常に把握。座標をメモして迷子防止。
